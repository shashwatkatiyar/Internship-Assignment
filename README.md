# Vitamin Mindmap App
This React app visualizes the hierarchy of vitamins in the human body using React Flow. Nodes can be expanded, collapsed, hovered, and selected. You can also add child nodes and download the current mindmap as a JSON file.

# Setup
Install dependencies and start the app:
```bash
npm start
```
# Features
- Click to view node details
- Double-click to expand/collapse children
- Hover tooltips with summaries
- Add child nodes via form
